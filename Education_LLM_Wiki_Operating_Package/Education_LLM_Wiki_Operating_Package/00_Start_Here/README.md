# Start Here

This is a lightweight Education LLM Wiki.

It is designed to turn raw learning materials, source notes, concepts, insights, and valuable learner questions into a reusable knowledge environment.

Use the vault through three loops:

1. **Source Loop** — process new materials into source notes and wiki pages.
2. **Query Loop** — answer learner questions and write valuable questions back into the wiki.
3. **Maintenance Loop** — lint and repair the wiki so it stays trustworthy, navigable, and fresh.

Read `AGENTS.md` first if you are an agent maintaining the system.
