# 上下文感知个性化伴学智能体

一句话介绍：这是一个嵌入学习平台的智能体，能读取当前课件、理解学生问题、诊断学习状态、生成小测、记录错因、安排复习，并把班级共性问题反馈给教师。

## 解决的实际问题

1. 学生不知道自己哪里不会。
2. 老师不知道学生集体卡在哪里。
3. PPT / 教材是静态内容，缺少互动。
4. 普通 LLM 回答无法追踪学习状态。
5. 学完之后缺少复习和巩固。
6. 个性化学习资源推荐缺少上下文。

## 智能体特征

1. 上下文感知：读取当前 PPT 页、前后页、平台大纲、教师讲稿和知识库证据。
2. 学生状态：维护 learner memory、薄弱概念、常见误区和小测历史。
3. 工具和知识库：在需要时调用本地 wiki / skill 检索，降低幻觉。
4. 行动计划：回答后可以生成小测、批改、复习任务和资源推荐。
5. 反馈闭环：学生作答会更新记忆，教师端能看到班级共性问题。

## 和普通 LLM 套壳的区别

普通 LLM 是“问一句，答一句”。

本系统是“感知当前页 → 判断困惑 → 解释 → 出题 → 批改 → 记录 → 复习 → 推荐资源 → 汇总教师洞察”。

## 真实落地场景

1. 课堂伴学。
2. 课后自学。
3. 在线课程。
4. 教师备课与反馈。
5. 个性化复习。
6. 学校知识库建设。

## MVP 与后续扩展

当前 MVP 使用本地 JSON 存储 learner memory 和 resource library，不做外部爬虫。后续可以接入学校 LMS、BilibiliAdapter、MoocAdapter、班级账号系统和真实教师看板。
